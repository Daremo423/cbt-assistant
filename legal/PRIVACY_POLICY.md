# Privacy Policy - CBT Assistant

## Data Collection

This app collects and processes:
- Audio recordings (processed in real-time)
- Transcribed text
- Identified cognitive distortions
- Usage analytics (anonymous)

## Data Processing

- Audio is processed using Deepgram's speech-to-text API
- Cognitive distortion detection occurs in real-time
- Audio files are deleted immediately after transcription (configurable)
- Transcripts are stored locally on your device only

## Third-Party Services

**Deepgram**: Provides speech-to-text transcription. Deepgram is SOC 2 Type II certified and HIPAA eligible. We have configured zero data retention with Deepgram.

## Your Rights

You have the right to:
- Access all data collected about you
- Delete all stored data at any time
- Export your data in a portable format
- Opt out of analytics collection

## Security

- All data transmission is encrypted (TLS 1.3)
- Local storage is encrypted
- No data is shared with third parties except as required for core functionality
- Regular security audits conducted
